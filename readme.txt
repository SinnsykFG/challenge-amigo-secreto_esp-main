Amigo Secreto Web App

Descripción

Esta aplicación web ha sido diseñada para facilitar la organización del juego del "Amigo Secreto" de manera rápida y sencilla. Permite a los usuarios registrar participantes, generar el sorteo de manera automática y enviar notificaciones con el resultado a cada persona de forma segura.

Características

Registro de participantes con nombres y correos electrónicos.
Algoritmo aleatorio para la asignación de los amigos secretos.
Notificación por correo electrónico de cada asignación (opcional).
Posibilidad de personalizar reglas como exclusiones (por ejemplo, evitar que parejas o familiares se asignen entre sí).

Interfaz amigable y de fácil uso.

Tecnologías Utilizadas

Frontend: HTML, CSS, JavaScript (posiblemente con frameworks como React o Vue.js).


Instrucciones de Instalación

Clonar el repositorio:

git clone https://github.com/usuario/amigo-secreto.git

Entrar en la carpeta del proyecto:

cd amigo-secreto

Instalar dependencias:

npm install

Configurar las variables de entorno:

Crear un archivo .env y definir los parámetros necesarios como base de datos, credenciales de correo, etc.

Ejecutar el servidor:

npm start

Uso

Acceder a la aplicación a través del navegador.

Ingresar los participantes del sorteo.

Realizar el sorteo y revisar las asignaciones.

Opcionalmente, enviar las notificaciones por correo.

Disfrutar del juego con los amigos o compañeros.
