Amigo Secreto App Web para Desafío Alura

Descripción

Esta aplicación web ha sido diseñada para facilitar la organización del juego del "Amigo Secreto" de manera rápida y sencilla.

Características

Registro de participantes con nombres.
Algoritmo para la asignación aleatoria de los amigos secretos.
Posibilidad de personalizar reglas como exclusiones, como no asignasrse a si mismo, o en caso de ser numero impar no quede alguien sin asignación o regalo.

Interfaz amigable y de fácil uso.

Tecnologías Utilizadas

Frontend: HTML, CSS, JavaScript.
Editor de código Visual Studio Code


Uso

Accede a la aplicación desde el repositorio de GitHub.

Ingresa los nombres de los participantes en el campo de texto y agrégalos a la lista.

Realiza el sorteo y verifica las asignaciones que se mostrarán en pantalla.

Disfruta del juego con tus amigos o compañeros.

Contribuciones

Las contribuciones son bienvenidas. Puedes enviar un pull request con mejoras o reportar problemas en la sección de issues del repositorio.
